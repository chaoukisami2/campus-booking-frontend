# Campus Booking Frontend

Frontend for the Campus Facilities Reservation backend (`capStoneBackend`).

## Features

- Register, login, logout
- Facility listing for all users
- Booking creation and cancellation
- Approve bookings for teacher/admin
- Facility creation for admin

## Backend API

This frontend expects the backend running at:

- `http://localhost:8080/api`

If your backend runs on another URL, update `API_BASE` in `src/App.jsx`.

## Run locally

```bash
npm install
npm run dev
```

Then open:

- `http://localhost:5173`
